# Stock-Portfolio-Tracker
A simple and beginner-friendly Stock Portfolio Tracker built using Python that helps users calculate the total value of their stock investments. The program allows users to enter stock names and quantities, then calculates the overall investment value using predefined stock prices stored in a dictionary.

This project is a great way to practice Python fundamentals such as dictionaries, user input handling, arithmetic operations, and optional file handling.

✨ Features

User inputs stock names and quantities

Uses a predefined dictionary for stock prices

Calculates total portfolio investment value

Simple console-based interaction

Option to save results in .txt or .csv format

Beginner-friendly and easy to understand


🛠️ Concepts Used

Dictionaries

User Input/Output

Basic Arithmetic Operations

File Handling (Optional)

Loops & Conditional Statements


🎯 Objective

To understand how basic financial tracking systems work while improving Python programming and logical thinking skills through a real-world inspired project.
